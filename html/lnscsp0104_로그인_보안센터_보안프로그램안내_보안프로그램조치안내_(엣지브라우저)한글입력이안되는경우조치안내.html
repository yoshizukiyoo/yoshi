보안프로그램조치안내

엣지 브라우저에서 한글 입력이 안되는 경우 조치 안내

뒤로가기

장애원인 : 구버전 엣지브라우저의 내부 오류로 키보드보안 프로그램의 정상적으로 동작하지 않아 한글 입력이 안되는현상

조치방법 : edge 브라우저 최신버전 재설치* 브라우저 확장 프로그램 추가 미설치 상태인 경우 추가 설치 필요

  1. 1. edge 브라우저 최신버전 설치

    1. 01 - 1마이크로소프트 홈페이지에서 새 Micofsoft Edge “다운로드” 클릭
      • 마이크로소프트 홈페이지 이동(https://www.microsoft.com.ko-kr.edge)
    2. 01 - 2“닫은 후 계속” 클릭
    3. 01 - 3설치 진행
    4. 01 - 4완료
    5. 01 - 5우체국 보험 홈페이지 접속(www.epostlife.go.kr)하여 로그인 클릭 시
      • 1. 로그인 화면으로 이동 시 → 거래 진행하여 한글 입력 여부 확인
      • 2. 키보드보안 보안프로그램 설치 화면으로 이동 시 → 다음 절차(2. 브라우저 확장 프로그램 설치) 진행

  2. 2. 브라우저 확장프로그램 설치

    1. 02 - 1브라우저 확장 기능 설치 팝업화면의 “등록하기” 클릭
      • 설치사유 : 최신버전 엣지 브라우저는 크롬 엔진을 사용하고 있어 크롬 extension 설치 필요
    2. 02 - 2화면 상단의 안내팝업 (다른 스토어의 확장 허용) 클릭
    3. 02 - 3“다른 스토어의 확장 허용” 팝업에서 “허용” 클릭
    4. 02 - 4“Chome에 추가” 클릭
    5. 02 - 5“TouchEn PC보안 확장”팝업에서 “확장추가” 클릭
    6. 02 - 6“동기화켜기” 클릭
    7. 02 - 7“확인” 클릭 후 인터넷뱅킹 홈페이지 접속 → 로그인 클릭
    8. 02 - 8정상설치 여부 확인
      • 로그인화면에서 아이디로그인의 아이디 입력란 클릭 시 화면 오른쪽 하단(키보드보안 동작 중(Ver 1.0.0.75)확인 가능

        - 단말 설정에 따라 아이콘이 나타나지 않을 수도 있으며 인터넷뱅킹 거래 정상 진행시 설치완료 상태임

      조치후에도 동일현상 반복 시
      키보드보안프로그램 고객센터(라온시큐어, 1644-4128)로 문의하여 주시기 바랍니다.
