안드로이드 OS

안드로이드 OS 이체안내

  2. 2. ‘계좌번호로 이체’ 탭
  3. 3. 정보입력 후 ‘다음’ 탭
  4. 4. 거래내역 확인 후 ‘다음‘ 탭
  5. 5. 받는 사람 확인 후 ‘예’ 탭
  6. 6. 정보확인 후 ‘이체하기’ 탭
  7. 7. 개인인증번호 입력
  8. 8.이체완료
