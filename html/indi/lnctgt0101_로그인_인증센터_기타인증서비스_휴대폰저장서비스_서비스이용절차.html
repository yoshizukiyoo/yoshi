휴대폰저장 서비스란?

인터넷 뱅킹에서 사용되는 인증서를 휴대폰에 저장해 두었다가, 본인의 인증서가 저장되어 있지 않은 PC에서(PC방 등) 휴대폰에 저장된 인증서를 불러와 인터넷뱅킹을 사용할 수 있도록 하는 유료서비스 입니다.

  • 1. 휴대폰에 인증서 저장
  • 2. 휴대폰 인증서 사용
  • 3. 인증서 관리

휴대폰 인증서 서비스 가입

  • PC → 휴대폰으로 무선인터넷을 통하여 인증서를 저장합니다.
  • 인증서는 안전한 방법으로 이동됩니다.
    휴대폰 인증번호를 이용한 휴대폰사용자 본인확인, PC와 휴대폰간의 End-to-End 암호화

인증서저장순서

  1. 휴대폰에 인증서 저장
  2. 휴대폰 인증서 사용
  3. 인증서 관리
  4. 서비스 체험하기

서비스사용이 가능한 휴대폰

  • 2002년 이후에 생산된 칼라액정 휴대폰은 일반적으로 사용이 가능하며, 휴대폰 게임을 다운로드 받을 수 있는 단말기면 본 서비스도 가능합니다. 이용 가능한 휴대폰에 대한 자세한 문의는 해당 통신사 콜센터로 문의하시기 바랍니다.
참고사항
  • 휴대폰에 설치된 인증서저장/전송 프로그램을 삭제하시거나 새로 다운로드를 받은 경우, 저장된 인증서가 삭제 됩니다.
    이 경우 PC -> 휴대폰으로 인증서를 다시 저장 하셔야 합니다.

인증서관리

휴대폰에 저장된 인증서 조회 / 삭제 방법

  • 휴대폰에 설치한 휴대폰인증서 서비스 UBIkey 실행 > 메뉴항목 중 ‘인증서관리‘ 선택
  • 인증서관리를 선택 - 저장된 인증서 목록이 나타납니다. 이중, 삭제를 원하는 항목을 선택하고 삭제 버튼을 선택 휴대폰인증서비스를 계속 이용하시려면 기간이 만료된 인증서나 재발급 등의 사유로 사용 불가한 인증서는 삭제하고 유효한 인증서를 다시 휴대폰에 저장하셔야 합니다.

휴대폰에 저장된UBIkey 프로그램 찾는 방법

통신사별 휴대폰에 저장된UBIkey 프로그램 찾는 방법 정보를 제공하고 있습니다.
피처폰 SKT 이용고객 휴대폰에서 Nate 선택> (m)playon(마법사보관함) 선택> 휴대폰 인증서 서비스(UBIkey)
피처폰 KT 이용고객 휴대폰에서 multipack/Show 선택> 다운로드센터/다운로드팩 > 휴대폰 인증서 서비스(UBIkey)
피처폰 LG U+ 이용고객 휴대폰에서 ez-i/LG U+ 선택 > DownTown > 휴대폰 인증서 서비스(UBIkey)
참고사항
  • 휴대폰인증서 APP 또는 VM(인증서관리프로그램)을 삭제할 경우, 인증서도 함께 삭제됩니다.
    인증서가 삭제된 경우, 하드디스크에 있는 인증서를 다시 저장하시면 됩니다.
