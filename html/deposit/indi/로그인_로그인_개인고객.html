스마트인증 로그인 이용 안내

  • 이용가능고객 : 우체국 스마트뱅킹에서 간편인증을 등록한 고객
  • 스마트인증 로그인 이용절차
    1. 핸드폰 번호 입력 후 '스마트인증 로그인' 선택
    2. 우체국 스마트뱅킹 푸시알림 선택
    3. 인터넷뱅킹 PC화면에 생성된 QR코드촬영 또는 인증요청번호 입력
    4. 본인인증(패턴, 지문, 핀번호 등)
    5. 인터넷뱅킹 '인증확인' 선택

스마트인증 이용방법

  1. 앱을 실행하여 [PC로그인/인증] 메뉴 선택
    • ㉮ 자동로그인 설정 고객은 취소버튼을 눌러 앱 홈화면으로 이동
    • ㉯ 홈화면 오른쪽 상단의 전체메뉴 버튼으로 메뉴를 펼침
    • ㉰ 보안인증센터 > PC로그인/인증 메뉴 실행
캡쳐화면 영역 (추후 작업 예정)

PASS인증서 로그인 이용안내

  • PASS인증이란?
    • 본인확인기관인 통신사 PASS앱을 이용해 보다 안전하고 간편하게 본인확인 및 로그인을 할 수 있는 서비스 입니다.
  • 이용가능고객 : 우체국 스마트뱅킹에서 간편인증 등록한 고객
  • PASS인증서 로그인 이용절차
    1. PC화면에서 성명, 휴대전화번호를 입력하여 로그인 요청
    2. 스마트폰의 PASS앱 실행/인증
      • ㉮ (PASS앱 Push확인)우체국 인터넷뱅킹 인증요청 내역 확인
      • ㉯ (본인인증)지문 또는 PASS앱 비밀번호 입력하여 본인인증
    3. 인증완료 후 PC화면의 [PASS인증확인] 버튼 클릭하여 로그인
