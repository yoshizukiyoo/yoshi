보안프로그램조치안내

엣지, 크롬 등 멀티브라우저에서 인터넷뱅킹 이용 시
키보드 보안 프로그램 설치가 안되는 경우(반복설치 등) 조치안내

뒤로가기

장애원인 : 브라우저 확장 프로그램 미설치로 인한 반복설치 유도

조치방법 : 브라우저 확장 프로그램 추가 설치

  1. 01우체국인터넷뱅킹 접속(www.epostlife.go.kr)하여 로그인 클릭 시 키보드보안 보안프로그램 설치화면으로 이동 됨

  2. 02브라우저 확장 기능 설치 팝업화면의  “등록하기”클릭

    • 설치사유 : 최신버전 엣지 브라우저는 크롬 엔진을 사용하고 있어 크롬 extension 설치 필요

  3. 03화면 상단의 안내팝업(다른 스토어의 확장 허용)클릭

  4. 04“다른 스토어의 확장 허용”팝업에서  “허용” 클릭

  5. 05“Chome에 추가”클릭

  6. 06“TouchEn PC보안 확장”팝업에서“확장추가”클릭

  7. 07“동기화켜기”클릭

  8. 08설치 완료 팝업창의 “확인”클릭 후 인터넷뱅킹 홈페이지 이동 후 로그인

    • 로그인화면으로 이동 시 설치 완료

  9. 09정상설치 여부 확인

    • 로그인화면에서 아이디로그인의 아이디 입력란 클릭 시 화면 오른쪽 하단(키보드보안 동작 중(Ver 1.0.0.75)확인 가능

      - 단말 설정에 따라 아이콘이 나타나지 않을 수도 있으며 인터넷뱅킹 거래 정상 진행시 설치완료 상태임

    조치후에도 동일현상 반복 시
    키보드보안프로그램 고객센터(라온시큐어, 1644-4128)로 문의하여 주시기 바랍니다.
