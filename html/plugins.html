							<span class="percent"></span>

						<script src="/resources/_common/plugins/easypiechart/jquery.easypiechart.min.js"></script>
						<script>					
							$('.easy-pie-chart').easyPieChart({
								barColor: '#999',
								trackColor: '#e7e7e7',
								scaleColor: false,
								lineCap: 'round',
								lineWidth: 5,
								size: 60,
								animate: 1000,
								onStep: function (from, to, percent) {
									$(this.el).find('.percent').text(Math.round(percent));
								},
							});
						</script>

  • canvas 기반이므로 별도의 CSS 파일이 존재하지 않으며 스크립트만으로 스타일 제어

							<div style="width: 100%; height: 300px;">
								<canvas id="myChart1">
