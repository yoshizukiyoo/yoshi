타이포 .txt_*

  • 스타일 미지정시 기본 폰트 사이즈 15px, 두께 200 (Light), 행간 1.5, 자간 0
  • 영문, 숫자 : 스포카 한 산스 / 한글 : 노토 산스 / 글꼴 기본 적용되어 있으므로 글꼴 개별 스타일링 불필요
